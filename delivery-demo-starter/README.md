# 🍃 CampusBite - 逢甲大學校園環保即時外送系統

> **2026 逢甲大學資訊專題小組 製作 ✕ 獨立開發完成**
> 
> CampusBite 是一個專為逢甲大學校園設計的高質感、環保友善的外送平台。系統採用優雅且兼具療癒感的**莫蘭迪（Morandi）色調**進行視覺設計，並針對校園生活場景，深度打造了優惠折扣、外送追蹤以及校園減碳三大核心模組。

---

## 🎨 專案核心特色

*   **極簡莫蘭迪視覺**：全站統一採用 `#8A9A86`（霧面鼠尾草綠）與 `#B5A69B`（溫暖燕麥灰）等低飽和度配色，打造清新優雅的文青風 UI。
*   **功能模組 A（優惠折扣）**：實作 `CAMPUS50` 與 `FCU85` 兩種情境的即時優惠券計算引擎。
*   **功能模組 B（即時外送追蹤）**：動態時間倒數、每 8 秒自動更新的狀態進度條，以及「外送員阿杰」擬真對話模擬器。
*   **功能模組 C（環保與快捷外送）**：支援「不索取一次性餐具」減碳設定，並提供福星宿舍、積學館、商學大樓等逢甲校內快捷外送點選擇。

## 👥 團隊成員與職責分工

我們團隊共有 **5 位成員**，在專案中各自擔任核心職能，從前端視覺、核心邏輯開發、專案管理到測試部署，皆有明確且專業的分工：

### 🧑‍💻 江柔萱 (Chiang Jou-hsuan) — 專案經理 (PM) & 核心邏輯工程師
*   **專案管理與架構設計**：負責規劃系統功能藍圖（Roadmap）、管控專案進度，並統籌小組的 Git 協作工作流。
*   **核心引擎開發**：主導 `js/app.js` 的核心 JavaScript 開發，包含購物車動態計算、優惠券折扣模組、外送狀態自動更新排程器，以及外送員自動對話回覆邏輯。
*   **跨頁面狀態同步**：設計購物車動態 Badge 顯示機制，確保全站（首頁、菜單、購物車、訂單狀態等頁面）資料完美同步。

### 🧑‍🎨 組員 B — UI/UX 設計師 & 前端視覺工程師
*   **全站視覺定義**：負責設計 CampusBite 的莫蘭迪色系調色盤，制定全站響應式（RWD）網格系統與圓角規範。
*   **CSS 樣式表實作**：編寫並維護 `css/base.css` 全域樣式，包含莫蘭迪色系變數（Variables）、卡片陰影效果，以及高度細緻的動畫效果。
*   **版面微調**：負責優化首頁、餐廳列表、會員中心與聯絡我們的排版，確保在電腦版與手機版均有極致的視覺美感。

### 👨‍💻 組員 C — 前端頁面工程師
*   **子頁面結構開發**：負責編寫網頁靜態 HTML 結構，包含 `pages/menu.html`（菜單）、`pages/cart.html`（購物車）與 `pages/order-status.html`（訂單追蹤）。
*   **表單與互動介面設計**：設計聯絡我們的回報表單（Contact Form）以及會員中心的個人資料卡片與歷史成就勳章區。
*   **API 與 JS 資料綁定**：與邏輯工程師合作，確保 HTML 的 DOM 元素與 JavaScript 的資料流正確綁定。

### 👩‍💻 組員 D — 測試與品質保證工程師 (QA)
*   **功能測試 (Functional Testing)**：針對優惠券折抵邏輯、免運費門檻計算、外送倒數計時器進行邊界條件與異常流程測試（例如輸入無效優惠券、重複折扣等）。
*   **跨瀏覽器相容性測試**：測試並修正網頁在 Chrome、Edge、Safari 等主流瀏覽器中的排版與顯示狀況。
*   **文字與細節校對**：負責校對全站菜單（16道料理）、環保提示文字、系統通知，避免錯別字，確保使用者體驗流暢。

### 👨‍💻 組員 E — 部署與版本控制工程師 (DevOps)
*   **Git 協作規範制定**：建立團隊的 Branch 分支管理策略，引導團隊落實 `main` 與 `dev` 分支的整併。
*   **GitHub 倉庫管理**：維護雲端 Repository，管理 Pull Request（PR）的審查與合併，確保專案程式碼無衝突且乾淨。
*   **部署上線**：負責設定並維護 GitHub Pages 靜態網頁線上預覽，確保期末簡報時網頁能穩定在線上呈現。

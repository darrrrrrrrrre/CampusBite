# 🍃 CampusBite - 逢甲大學校園環保即時外送系統

> **2026 逢甲大學資訊專題小組 製作 ✕ 獨立開發完成**
> 
> CampusBite 是一個專為逢甲大學校園設計的高質感、環保友善的外送平台。系統採用優雅且兼具療癒感的**莫蘭迪（Morandi）色調**進行視覺設計，並針對校園生活場景，深度打造了優惠折扣、外送追蹤以及校園減碳三大核心模組[cite: 1]。

---

## 🎨 專案核心特色

*   **極簡莫蘭迪視覺**：全站統一採用 `#8A9A86`（霧面鼠尾草綠）與 `#B5A69B`（溫暖燕麥灰）等低飽和度配色，打造清新優雅的文青風 UI。
*   **功能模組 A（優惠折扣）**：實作 `CAMPUS50` 與 `FCU85` 兩種情境的即時優惠券計算引擎[cite: 1]。
*   **功能模組 B（即時外送追蹤）**：動態時間倒數、每 8 秒自動更新的狀態進度條，以及「外送員阿杰」擬真對話模擬器[cite: 1]。
*   **功能模組 C（環保與快捷外送）**：支援「不索取一次性餐具」減碳設定，並提供福星宿舍、積學館、商學大樓等逢甲校內快捷外送點選擇[cite: 1]。

## 👥 團隊成員、職責分工與具體改動

我們團隊共有 **5 位成員**。在本次系統大升級中，每位組員除了各司其職，更針對各自負責的模組進行了深度的程式碼撰寫與改動：

### 🧑‍💻 江柔萱 (Chiang Jou-hsuan) — 專案經理 (PM) & 核心邏輯工程師
*   **專案管理與架構設計**：負責規劃系統功能藍圖（Roadmap），定義三大核心功能模組的規格與互動流程，並統籌小組的 Git 協作工作流。
*   **具體程式改動與實作（`js/app.js` 核心開發）**：
    *   **資料結構設計**：設計並維護 16 道精美菜單資料庫（包含健康輕食、校園炸物、午茶甜點三大類別與價格、圖片、描述屬性）[cite: 1]。
    *   **優惠券折扣引擎**：撰寫 `applyCoupon()` 邏輯，實現輸入 `CAMPUS50`（現折 50 元）或 `FCU85`（全單 85 折）的動態金額試算與無效優惠券的阻擋機制[cite: 1]。
    *   **購物車動態管理**：開發購物車的 `addToCart()`、`updateQuantity()`、`renderCart()` 與 `updateCartBadge()` 功能，實現全站導覽列購物車 Badge 數量的即時同步[cite: 1]。
    *   **動態外送模擬排程器**：實作 `setInterval` 定時器，以 8 秒為週期推進訂單狀態（商家接單 ➔ 精心烹調 ➔ 外送飛奔 ➔ 美味送達），並動態計算寬度與倒數時間[cite: 1]。
    *   **外送對話機器人**：設計 `sendChatMessage()` 函數，並透過 `setTimeout` 實作 1.5 秒後外送員「阿杰」的擬真自動回覆邏輯[cite: 1]。

### 🧑‍🎨 劉宏彥 — UI/UX 設計師 & 前端視覺工程師
*   **全站視覺定義**：負責設計 CampusBite 的莫蘭迪色系調色盤（包含深綠、燕麥灰、玫瑰粉、暖白等），制定全站響應式（RWD）網格系統。
*   **具體程式改動與實作（`css/base.css` 視覺大翻修）**：
    *   **CSS 變數系統**：在 `:root` 宣告莫蘭迪專屬色碼變數（如 `--morandi-green: #8A9A86`、`--morandi-oat: #B5A69B` 等），大幅提升樣式可維護性。
    *   **精美卡片與陰影效果**：改寫 `.card` 與 `.tracking-card` 樣式，加入優雅的邊框（`border`）、圓角（`border-radius`）與極輕量的陰影（`box-shadow`），擺脫傳統刺眼的飽和色調。
    *   **動態進度條動畫**：設計 `.delivery-visual`、`.progress-bar-fill` 的寬度漸變動畫與 `.progress-step.active` 的放大縮放特效，讓訂單狀態推移更加自然。
    *   **RWD 斷點微調**：針對手機裝置（`@media (max-width: 768px)`）重寫導覽列與首頁網格，確保行動端視覺體驗流暢不跑版。
